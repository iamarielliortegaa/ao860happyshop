## üöÄ **How Images Display in App**

### Product Cards:
- **Main View**: 250x180px (large, clear display)
- **Cart View**: 80x60px (thumbnail)
- **Fallback**: If image missing, placeholder used

### Image Loading:
```java
// From ShoppingCenterView.java
String imgSrc = null;
if (p.getImageFilename() != null && !p.getImageFilename().isBlank()) {
    Path fp = StorageLocation.imageFolderPath.resolve(p.getImageFilename());
    if (Files.exists(fp)) imgSrc = fp.toUri().toString();
}
if (imgSrc == null) {
    // Fallback to placeholder
    URL res = getClass().getResource("/imageHolder.jpg");
    imgSrc = res == null ? "" : res.toExternalForm();
}
```
